# Sales-Dashboard
Sales performance dashboards (Excel &amp; Power BI)

## 1. Description
Project ini bertujuan untuk menganalisis performa penjualan berdasarkan kategori produk, Bulan, dan Tahun guna memperoleh insight yang dapat membantu pengambilan keputusan bisnis.

## 2. Objectives
- Mengetahui total revenue, jumlah order, dan jumlah customer
- Menganalisis performa penjualan berdasarkan kategori dan musim
- Mengidentifikasi tren pembelian pelanggan
- Menentukan kategori dengan kontribusi revenue tertinggi

## 3. Dataset
- Data transaksi penjualan
- Kolom: Order ID, Customer ID, Category, Purchase Amount (USD), Season, Payment Method, Order Status
- Format data: CSV
- Sumber data: Dataset simulasi

## 4. Tools & Technologies
- Power BI
- SQL (MySQL)
- Microsoft Excel

## 5. Data Cleaning & Preparation
- Menghapus data duplikat
- Menangani missing values
- Menyesuaikan format data
- Memfilter data dengan status order "Completed"

## 6. Analysis & Methodology
- Agregasi revenue menggunakan fungsi SUM
- Perhitungan jumlah customer unik
- Analisis tren musiman
- Perbandingan performa penjualan antar kategori dan metode pembayaran

## 8. Key Insights
- Kategori tertentu memberikan kontribusi revenue tertinggi
- Terjadi peningkatan penjualan pada musim tertentu
- Metode pembayaran tertentu paling dominan digunakan pelanggan

## 9. Conclusion
Hasil analisis menunjukkan bahwa fokus pada kategori dengan performa terbaik serta strategi promosi musiman dapat meningkatkan revenue perusahaan.
